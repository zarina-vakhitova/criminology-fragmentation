Do Criminologists Read Each Other’s Work?

Interactive cluster specification explorer — companion to the paper by Zarina I. Vakhitova (Monash University). Move journals between intellectual traditions and watch the fragmentation ratio recompute in real time.
Vakhitova, Z. I. (2026). Do criminologists read each other’s work? A longitudinal analysis of cross-citation patterns, 1960–2025.
Data: Web of Science Core Collection, 23 journals, 1973–2025. Size-adjusted within/between ratio (3-year smoothed).
